# Skills

Agent skills thats i use everyday.

By [Fathin Afif](https://fathin.dev/) · [GitHub](https://github.com/fathinafiff)

## Available Skills

### Engineering

- **[btca-local](./skills/btca-local/SKILL.md)** — Search git repos locally. Clone repos to `~/.btca/agent/sandbox` and search them for answers with code examples.

### Productivity

- **[grill-me](./skills/grill-me/SKILL.md)** — Get relentlessly interviewed about a plan or design until every branch of the decision tree is resolved. Writes notes to `dev-docs/`.
- **[phased-plan](./skills/phased-plan/SKILL.md)** — Create a practical phased implementation plan after requirements are clear, then execute one phase at a time with approval gates.

### Content

- **[medium-personal-article](./skills/personal/medium-personal-article/SKILL.md)** — Write personal, narrative-driven technical articles for Medium in a relatable storytelling style.

### Social Media

- **[threads-api](./skills/sosmed/threads-api/SKILL.md)** — Publish and manage Meta Threads posts programmatically via the Threads API. Includes scripts for autoposting.
